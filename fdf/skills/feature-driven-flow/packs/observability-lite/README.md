# Pack: observability-lite

Adds a small amount of additional workflow observability without changing core behavior.

Contents:

1. Rule: `observability-lite`
2. Profile: `observability-lite-overlay` (optional overlay)

How to enable:

1. Add `observability-lite` to `packs.enabled` in `.codex/feature-driven-flow/settings.json`.
2. Optionally select `observability-lite-overlay`.
