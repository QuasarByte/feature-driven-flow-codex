# Clarify Questions Template

Use this structure in Phase 3 (Clarify).

## Questions

1. Scope:
   - What is explicitly in scope?
   - What is explicitly out of scope?
2. Behavior:
   - What should happen on invalid input?
   - What should happen on partial failure?
3. Rule-driven concerns:
   - What concerns are required by active rules (for example compatibility, performance, security, operations, data)?
4. Interfaces and dependencies:
   - What contracts or integrations must remain stable?
5. Acceptance and constraints:
   - What acceptance criteria and constraints are mandatory?

## Assumptions

1. Assumption A
2. Assumption B

## Decision Options (Use If User Delegates)

1. Option 1: <choice> (`recommended`)
2. Option 2: <choice>
3. Option 3: <choice>

## Rationale

1. One-line tradeoff explanation for the recommended option.

## Confirmation Prompt

Please answer these questions. If you want delegation, reply with option number (or custom answer) for each delegated decision.
